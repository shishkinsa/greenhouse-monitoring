## BR-01: Единый мониторинг всех теплиц в масштабе компании

### Метаданные

| Атрибут | Значение |
|---------|----------|
| **ID** | BR-01 |
| **Тип** | Бизнес |
| **Статус** | Утверждено |
| **Приоритет** | Критический |
| **Автор** | Команда проекта |
| **Владелец** | Команда проекта |
| **Создано** | 2025-03-14 |
| **Обновлено** | 2025-03-14 |
| **Версия** | 1.0 |

### Описание

Обеспечить единую информационную систему для мониторинга свыше 1000 теплиц, распределённых по всей стране, с единой точкой входа для сотрудников и инженеров компании.

### Детальное описание

- **Бизнес-ценность:** централизованный контроль состояния тепличного хозяйства, оперативное принятие решений на основе актуальных данных со всех объектов.
- **Метрики успеха:** все теплицы доступны через веб-приложение; сотрудники (~1000) и инженеры (~50) могут получать доступ к данным в соответствии со своими ролями.
- **Влияние на пользователей:** единый интерфейс вместо разрозненных решений; возможность быстрого переключения между теплицами и объектами.

| **Автор** | Команда проекта |
| **Владелец** | Команда проекта |
| **Создано** | 2025-03-14 |
| **Обновлено** | 2025-03-14 |
| **Версия** | 1.0 |

### Описание

Сотрудники компании должны иметь возможность выбирать теплицу и видеокамеру, смотреть видеопоток в реальном времени с наложением в нижнем правом углу актуальных показаний датчиков (температура, влажность, кислотность почвы).

### Детальное описание

- **Бизнес-ценность:** визуальный контроль состояния теплицы в сочетании с числовыми показателями датчиков для оперативной оценки ситуации.
- **Метрики успеха:** задержка отображения видео и данных не более 30 секунд; отображение всех трёх показателей датчиков в реальном времени.
- **Влияние на пользователей:** сотрудники получают полную картину по выбранной теплице без необходимости переключаться между разными системами.

### Описание

Инженеры компании должны иметь возможность задавать арифметико-логические выражения по показателям датчиков, выбирать теплицы и временные интервалы, просматривать архивное видео с камер и динамику изменения показателей датчиков по отобранным событиям.

- **Влияние на пользователей:** инженеры могут эффективно искать и анализировать события без ручного просмотра больших объёмов данных.
