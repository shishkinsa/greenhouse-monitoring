### Описание

### Детальное описание

ИС проектируется с учётом количества объектов (1000+ теплиц) и их географического распределения. Это влияет на требования к производительности, хранению данных и отказоустойчивости. Ссылка на NFR-04 (масштабируемость).

## BR-R02: Состав оборудования в теплице

### Метаданные

| Атрибут | Значение |
|---------|----------|
| **ID** | BR-R02 |
| **Тип** | Бизнес |
| **Статус** | Утверждено |
| **Приоритет** | Критический |
| **Автор** | Команда проекта |
| **Владелец** | Команда проекта |
| **Создано** | 2025-03-14 |
| **Обновлено** | 2025-03-14 |
| **Версия** | 1.0 |

### Описание

В каждой теплице установлены: три датчика (температура, влажность, кислотность почвы), один контроллер, собирающий данные с датчиков, и до 5 цифровых видеокамер.

### Детальное описание

Контроллер и камеры подключаются к вычислительной сети. Система должна корректно отображать данные от всех трёх типов датчиков и поддерживать выбор любой из камер (от 1 до 5) в рамках одной теплицы.

| **Владелец** | Команда проекта |
| **Создано** | 2025-03-14 |
| **Обновлено** | 2025-03-14 |
| **Версия** | 1.0 |

### Описание

Контроллер передаёт информацию о показателях датчиков по протоколу MQTT при инициализации контроллера и при изменении значений показателей.

### Детальное описание

ИС должна принимать и обрабатывать сообщения MQTT от контроллеров. Моменты отправки данных определяются оборудованием (старт и изменение показаний). Система должна обеспечивать отображение актуальных данных в режиме реального времени с учётом ограничения по задержке (BR-R05, NFR-01).
