# Диаграммы архитектуры

## Назначение

Эта папка содержит все диаграммы архитектуры системы в формате "как код" (diagrams as code).
Диаграммы автоматически генерируются в PNG для встраивания в документацию.

## Правила именования

### Формат

[номер]-[тип]-[описание].[расширение]

### Примеры

| Файл | Описание |
|------|----------|
| `01-c4-system-context.c4` | Контекст системы (C4) |
| `10-drawio-er-database.drawio` | ER-диаграмма БД |
| `20-plantuml-state-order.puml` | Состояния заказа |

### Номера по типам

| Диапазон | Тип |
|----------|-----|
| 01-09 | C4 модели |
| 10-29 | Draw.io |
| 30-49 | PlantUML |
| 50+ | Резерв |

## Типы диаграмм

### C4 модели (LikeC4)

| Уровень | Файл | Описание |
|---------|------|----------|
| Level 1 | `01-system-context.c4` | Системный контекст |
| Level 2 | `02-containers.c4` | Контейнеры |
| Level 3 | `03-components-api.c4` | Компоненты API |
| Level 4 | `04-code.c4` | Код (опционально) |

### Draw.io

| Категория | Файл | Описание |
|-----------|------|----------|
| ER | `10-database-schema.drawio` | Схема БД |
| Deployment | `20-aws-infra.drawio` | Инфраструктура |

### PlantUML

| Категория | Файл | Описание |
|-----------|------|----------|
| State | `50-order-state.puml` | Жизненный цикл заказа |
| Algorithm | `60-rate-limiting.puml` | Rate limiting |
| Sequence | `70-auth-sequence.puml` | Sequence аутентификации |

## Инструменты

### Необходимое ПО

| Инструмент | Установка | Для чего |
|------------|-----------|----------|
| **LikeC4** | `npm install -g likec4` | C4 модели |
| **PlantUML** | `brew install plantuml` | UML диаграммы |
| **Draw.io** | Расширение VS Code | Draw.io диаграммы |
