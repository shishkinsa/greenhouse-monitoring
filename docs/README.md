# Документация по системе

## Структура папок

| Раздел | Путь | Содержит |
|--------|------|----------|
| **Код** | `/src` | Исходный код приложения |
| **Требования** | `/docs/requirements/` | Бизнес, функциональные, нефункциональные требования |
| **Стандарты** | `/docs/standards/` | Правила и соглашения |
| **Архитектура** | `/docs/architecture/` | ADR и диаграммы |
| **API** | `/docs/openapi/` | OpenAPI спецификации |
| **Артефакты** | `/docs/public/` | Сгенерированные PNG |
