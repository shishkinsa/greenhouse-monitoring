# Документация по системе

В папке `docs/` собрана проектная документация: требования, стандарты кода и Git, архитектурные решения и спецификации API. Навигация по разделам — через таблицу ниже.

## Структура папок

| Раздел | Ссылка | Содержит |
|--------|--------|----------|
| **Код** | [src/](../src/) | Исходный код приложения |
| **AI** | [ai/](ai/) | Детальные инструкции для AI |
| **Требования** | [requirements/](requirements/) | Бизнес, функциональные, нефункциональные требования |
| **Стандарты** | [standards/](standards/) | Правила и соглашения |
| **Архитектура** | [architecture/](architecture/) | ADR и диаграммы |
| **API** | [openapi/](openapi/) | OpenAPI спецификации |
| **Артефакты** | [public/](public/) | Сгенерированные PNG |

## Стандарты (standards/)

В разделе [standards/](standards/) собраны правила именования (C#, TypeScript, React, PostgreSQL), общие стандарты кодирования и процесс работы с Git (Git Flow). Полный перечень документов и краткое описание каждого — в [standards/README.md](standards/README.md).
